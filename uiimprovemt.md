Design a developer tool UI for a Logic Gate Simulator.

Style:
- Inspired by VS Code and Figma canvas
- Dark industrial theme (no gradients, no glassmorphism)
- Functional, minimal, dense layout

Layout:
- Top toolbar (run, reset, save)
- Left sidebar (gate library)
- Center canvas (grid background)
- Right panel (properties)

UI Requirements:
- Grid-based canvas
- Drag-and-drop gates
- Subtle hover states
- No flashy colors

Color System:
- Background: deep charcoal
- Panels: slightly lighter
- Signals:
  - Green for HIGH
  - Red for LOW
  - Gray for inactive

Focus on usability and clarity, not decoration.

Output clean Tailwind or React components.
