LOGIC GATES

AND Gate

The AND gate outputs a "1" (HIGH) only when all its inputs are "1" (HIGH). If any input is "0" (LOW), the output is "0" (LOW).

OR Gate

The OR gate outputs a "1" (HIGH) if at least one of its inputs is "1" (HIGH). It outputs "0" (LOW) only when all inputs are "0" (LOW).

NOT Gate

The NOT gate, also known as an inverter, has only one input. It outputs the opposite of its input. If the input is "1" (HIGH), the output is "0" (LOW), and vice versa.

XOR Gate

The XOR (Exclusive OR) gate outputs a "1" (HIGH) only when its inputs are different. If both inputs are the same (both "0" or both "1"), the output is "0" (LOW).

NAND Gate

The NAND gate is a combination of a NOT gate and an AND gate. It outputs a "0" (LOW) only when all its inputs are "1" (HIGH). Otherwise, the output is "1" (HIGH).

NOR Gate

The NOR gate is a combination of a NOT gate and an OR gate. It outputs a "1" (HIGH) only when all its inputs are "0" (LOW). If any input is "1" (HIGH), the output is "0" (LOW).
